WELCOME TO PERF-TRACE!
Perf Trace is an application that:
tracks user's activity on this page (Javascript) sends them to the server (REST, Java) the server every minute collects data and groups statistics (Spring scheduling) and provides those statistics through the REST api. This you can see through this app (Angular)
Therefore you can trace your activity and see it here at once!
To go to the perf-trace client app click HERE on pawelczyk.biz.pl/perf-trace
The perf-trace-server app is avaible on pawelczyk.biz.pl:8081. It listens for the perf-trace REST requests for logging activity as well as for displaying this page ;)
